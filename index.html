My Projects

Employee Management Application for a Hospital

This project involves developing a hospital employee management application with role-based access control. The system will include CRUD functionalities (Create, Read, Update, Delete) for managing employees and a feature that assigns specific authorizations to staff members. These authorizations will determine access to the hospital’s medication stock, ensuring secure and efficient resource management. The application aims to enhance hospital operations by providing a structured and controlled access system.

Tech Stack: Java, JDBC, Apache Maven, JavaFx, JUnit, PostgreSQL, Multithreading, Git, Batch, Jenkins, CSS

Shift Scheduling and Management Application

This project focuses on developing an application to organize and schedule morning and evening shifts for employees, providing a clear overview of each person's availability. The system includes an algorithm that assigns guard locations based on employees’ capabilities while managing constraints such as replacing absent or sick staff members. Additionally, the application will incorporate a probability-based algorithm to predict and handle potential absences, ensuring smooth and efficient shift management..

Tech Stack: Java, SpringBoot, PostgreSQL, Jenkins, Git, React, TailwindCSS, React, Nodejs, Javascript, HTML
